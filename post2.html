IPL-Win Probabaility Predictor

I worked on an end-to-end machine learning project focused on predicting win probabilities in the Indian Premier League (IPL). I collected data on IPL matches, including team performance, player statistics, and match outcomes, and preprocessed it to prepare for analysis.

After selecting key features like team rankings and player form, I chose logistic regression as the primary model for predicting win probabilities due to its simplicity and interpretability. I trained the model on historical IPL data and evaluated its performance using metrics like accuracy and F1 score.

To make the predictions accessible, I created a dashboard using the Streamlit framework. This dashboard visualized the model's predictions and provided insights into match outcomes, serving as a valuable tool for IPL enthusiasts and analysts.

Overall, this project showcased the application of machine learning in predicting win probabilities in cricket matches, highlighting the potential of data-driven insights in sports analytics.
